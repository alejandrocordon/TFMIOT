Inicio Rapido

Requisitos: Python 3.9+, pip. GPU opcional pero recomendada para training.
  1. Instalar MLForge 
    pip install -e .[all]
  2. Crear un nuevo proyecto 
    mlforge new mi_clasificador --task classification
  3. Entrenar el modelo 
    mlforge train --config config.yaml
  4. Exportar a formatos de despliegue 
    mlforge export --config config.yaml --formats all
  5. Desplegar en cualquier plataforma 
    mlforge deploy android --model model.tflite --labels "rosa,tulipan,girasol"
