Patron de Registro de Modelos

MLForge usa un patron de decoradores para registrar modelos automaticamente:

# Registrar un modelo PyTorch
from mlforge.models.registry import register_pytorch

@register_pytorch("mi_modelo")
def mi_modelo(num_classes: int, pretrained: bool):
    # Tu arquitectura custom aqui
    model = ...
    return model

# Ahora disponible en config.yaml:
# model:
#   architecture: mi_modelo
El registro de modelos permite anadir arquitecturas custom sin modificar el core
