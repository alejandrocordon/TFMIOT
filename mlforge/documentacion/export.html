Deploy Templates

MLForge genera proyectos completos para cada plataforma con mlforge deploy <target>.

📱

Android

Kotlin + CameraX + TFLite

  • MainActivity con TFLite Interpreter
  • CameraX ImageAnalysis en tiempo real
  • Template vars: labels, input_size
  • build.gradle.kts con dependencias
mlforge deploy android \
  --model model.tflite \
  --labels "gato,perro"
🍎

iOS

SwiftUI + CoreML + Vision

  • ContentView con ImagePicker
  • VNCoreMLRequest para inferencia
  • Apple Neural Engine support
  • Interface drag & drop
mlforge deploy ios \
  --model model.mlpackage \
  --labels "gato,perro"
🌐

Web

HTML + ONNX Runtime Web

  • Drag & drop de imagenes
  • ONNX Runtime desde CDN
  • WebGL backend GPU
  • Barras de confianza top-5
mlforge deploy web \
  --model model.onnx \
  --labels "gato,perro"
🤖

Edge (RPi + Coral)

Python + PyCoral/TFLite Runtime

  • Inferencia con Edge TPU o CPU
  • Modo camara en vivo (OpenCV)
  • Modo imagen individual
  • FPS overlay en video
mlforge deploy edge \
  --model model_edgetpu.tflite \
  --labels "gato,perro"
